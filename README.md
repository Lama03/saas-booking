# saas-booking

🟣 Super Admin — manages the whole system
🟠 Tenant Admin — manages their own business (services, bookings)
🟢 Client/Customer — books services


🟣 Super Admin Pages (System Owner)
Page	Description
🔐 Login Page	Secure login for super admin
🏠 Dashboard	Overview of tenants, usage, revenue
🏢 Manage Tenants	View/add/edit/delete tenants (business owners)
👥 Manage Users	View all system-wide users
💼 Subscriptions/Plans	Manage pricing tiers (Free, Pro, Premium)
⚙️ System Settings	Global settings (logo, currency, language, etc.)
💬 Support/Contact Messages	View messages submitted by tenants or users

🟠 Tenant Admin Pages (Business Owner)
Page	Description
🔐 Tenant Login	Separate login page for tenants
🏠 Dashboard	Overview: total bookings, services, revenue
📋 Booking Management	View, accept, cancel bookings
🛠 Manage Services	Add/edit/delete services
🕐 Availability	Set working hours / available time slots
👨‍👩‍👧‍👦 Staff Management	Add team members (e.g., receptionists, service providers)
👥 Client List	View client info, past bookings
💸 Payment Records	View payments made through the platform
📄 Booking Calendar	View bookings in calendar format
⚙️ Business Settings	Change business name, logo, address, contact info

🟢 Client/Customer Pages (End-User)
Page	Description
🔐 Client Login/Register	User account creation and login
🏠 Home Page	Landing page with business intro
🔍 Service List	Browse services offered by the tenant
📅 Booking Page	Select service, date, time, and book
💳 Checkout/Payment	Make payment for the booking (Stripe, PayPal, etc.)
📜 My Bookings	See upcoming and past bookings
⭐ Review Page	Leave a rating/review for service
👤 Profile Page	View/update personal details, change password

🧠 Shared/Global Pages (for all roles or public)
Page	Description
🌐 Landing Page	Public site for marketing
📞 Contact Page	Contact form
❓ FAQ	Frequently Asked Questions
🧾 Terms & Privacy	Legal pages
🔐 Forgot Password	Reset login credentials
